### Workstream Isolation and Handoff Rules

The four workstreams are functionally connected, but each owner must be able to implement assigned work without editing another owner's files. A downstream task may use a versioned contract or mock while its prerequisite is unfinished, but it cannot be completed until the real upstream handoff and integration tests pass. The AIO4 Lead is accountable for integration and acceptance decisions but does not bypass specialist file ownership.

| Owner | Exclusive write scope | Consumed by other members through |
