Có 2 nguồn đầu vào chính:

- `Prometheus polling`: AIOps chủ động query Prometheus theo các query ID đã đăng ký sẵn.
- [[Grafana hard-rule webhook]] gửi alert event sang AIOps khi rule hard SLO firing/resolved.

Điểm quan trọng: AIOps không tự tạo query tùy tiện từ alert text. Nó chỉ dùng query đã đăng ký và validate trong config.

Ví dụ:

```text
Prometheus -> checkout_error_ratio_24h
Prometheus -> db_backend_usage
Grafana -> checkout SLO breach firing event
```
