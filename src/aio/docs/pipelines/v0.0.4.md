**Runtime Pipeline**
```mermaid
flowchart TD
    A["FastAPI / API caller"] --> B["run_static_pipeline(request, settings)"]
    B --> C["Load Settings from .env"]
    B --> D["Load RuntimeConfig from config/runtime.json"]
    D --> E["build_detectors(runtime_config, settings)"]
    B --> F["SQLiteIncidentStore"]
    B --> G["AiopsPipeline.run_once(metric_series)"]

    G --> H["Collector.collect()"]
    H --> I["Normalizer.normalize()<br/>CDO label/unit/window aliases"]
    I --> J["QualificationGate.evaluate()"]
    J --> K["FeatureBuilder.build()"]
    K --> L["DetectorEngine.evaluate()"]

    L --> L1["ThresholdDetector"]
    L --> L2["DependencyDetector"]
    L --> L3["NoDataDetector"]

    L1 --> M["CandidateEvent[]"]
    L2 --> M
    L3 --> M

    M --> N["Correlator.correlate()"]
    N --> O["Enricher.enrich(features)"]
    O --> P["SQLiteIncidentStore.upsert()"]
    P --> Q["Incident[]"]

    Q --> R["NotificationBuilder.build()"]
    Q --> S["PolicyEngine.proposal_for()"]
    S --> T["PolicyEngine.evaluate()"]
    T --> U["PolicyDecision[]"]

    Q --> V["VerificationEngine.verify(features)"]
    V --> W["VerificationResult[]"]

    G --> X["V001 RCA"]
    X --> Y["RcaResult"]

    Q --> Z["Remediation Strategy"]
    Y --> Z
    Z --> ZA["RemediationDecision[]"]
    ZA --> ZB["RemediationAuditLog.append()"]

    W --> ZC{"Verification recovered?"}
    ZA --> ZC
    Y --> ZC
    ZC -->|yes| ZD["IncidentHistoryStore.append_success()"]
    ZC -->|no| ZE["Do not update history"]

    R --> OUT["PipelineResult"]
    U --> OUT
    W --> OUT
    Y --> OUT
    ZA --> OUT
```

**Anomaly Detection Pipeline**
```mermaid
flowchart TD
    A["MetricSeries[] + optional logs"] --> B["V001AnomalyEngine.evaluate()"]

    B --> C["Metric detectors"]
    C --> C1["RobustDriftDetector"]
    C --> C2["EwmaStlDetector"]
    C --> C3["ServiceIsolationForestDetector"]

    C1 --> D["Raw metric findings"]
    C2 --> D
    C3 --> D

    D --> E["_weighted_sum()"]
    E --> E1{"score >= weighted_score_threshold?"}
    E1 -->|no| E2["Drop weak finding"]
    E1 -->|yes| E3["AnomalyFinding algorithm=weighted_sum"]

    B --> F["LogTemplateMetricBuilder.build(logs)"]
    F --> G["Log metric series"]
    G --> H["EWMA + IsolationForest on log metrics"]
    H --> I["_weighted_sum(log findings)"]

    I --> J{"same service and near metric anomaly?"}
    J -->|no| J1["Drop log-only finding"]
    J -->|yes| J2["Keep correlated log finding"]

    E3 --> K["Metric findings"]
    J2 --> K

    K --> L["_suppress_busy_infra()"]
    L --> M{"CPU or disk IO metric?"}
    M -->|no| Z["Keep anomaly"]
    M -->|yes| N{"request_rate increased?"}
    N -->|no| Z
    N -->|yes| O{"latency/error/ready_pods/memory also abnormal?"}
    O -->|yes| Z
    O -->|no| P["Suppress as busy load"]

    Z --> R["Final AnomalyFinding[]"]
    P --> R
```

**RCA Pipeline**
```mermaid
flowchart TD
    A["metric_series + incidents"] --> B["_run_v001_rca()"]
    B --> C["Extract log messages from incident evidence"]
    C --> D{"RCA enabled and metric_series or logs exist?"}
    D -->|no| E["RcaResult empty"]
    D -->|yes| F["V001AnomalyEngine.evaluate(metric_series, logs)"]

    F --> G["AnomalyFinding[]"]
    G --> H["V001RcaEngine.rank(findings, metric_series, top_k)"]

    H --> I["Canonicalize service names"]
    I --> J["Drop excluded root causes"]
    J --> K{"Any root findings?"}
    K -->|no| L["RcaResult anomalies only"]

    K -->|yes| M["Ranker scores"]
    M --> M1["GraphTraversalRca.rank_services()"]
    M --> M2["_earliest_drift_scores()"]
    M --> M3["_correlation_scores()"]

    M1 --> N["_weighted_rrf()"]
    M2 --> N
    M3 --> N

    J --> O["Collect root cause metrics"]
    O --> O1["Weighted anomaly metrics"]
    O --> O2["Robust drift metrics"]
    O1 --> P["Apply metric aliases"]
    O2 --> P

    N --> Q["Sort services by score"]
    P --> Q
    Q --> R["RootCauseCandidate top-K"]
    R --> S["RcaResult anomalies + root_causes"]
```

**RCA Post-processing**
```mermaid
flowchart TD
    A["RcaResult"] --> B{"Root cause exists?"}
    B -->|no| C["Use incident conclusion only"]
    B -->|yes| D["Log AIOPS_CONCLUSION source=rca"]

    D --> E{"Root service already has incident in this run?"}
    E -->|yes| F["Do not create synthetic incident"]
    E -->|no| G["Create CandidateEvent detector=rca_root_cause"]
    G --> H["SQLiteIncidentStore.upsert()"]
    H --> I["Synthetic SEV2 RCA incident + notification"]

    D --> J{"root.score >= suppress_min_root_score?"}
    J -->|no| K["Do not register new blast-radius suppression"]
    J -->|yes| L["Compute blast radius from topology"]
    L --> M["Register active root cause TTL"]
    M --> N["Suppress child notifications in same blast radius"]

    A --> O["Append RCA history JSONL"]
```

**Remediation Pipeline**
```mermaid
flowchart TD
    A["Incident + RcaResult"] --> B["RemediationFeatureExtractor.extract()"]

    B --> B1["affected_services = incident.service + dependency + RCA services"]
    B --> B2["log_signatures = event.reason"]
    B --> B3["metric_ratios = value / threshold"]

    B1 --> C["IncidentFeatures"]
    B2 --> C
    B3 --> C

    C --> D["IncidentHistoryStore.load()"]
    C --> E["HistoryRetriever.top_matches()"]

    D --> E
    E --> E1["similarity = service jaccard + log jaccard + metric similarity"]

    E1 --> F["RemediationDecisionEngine.decide()"]
    G["ActionCatalog.load()"] --> F

    F --> H{"similarity < OOD threshold?"}
    H -->|yes| I["fallback page_oncall"]
    H -->|no| J["Vote actions from matched history"]

    J --> K{"action exists in catalog?"}
    K -->|no| I
    K -->|yes| L["Compute confidence + expected_cost"]

    L --> M{"Guardrails pass?"}
    M -->|no| I
    M -->|yes| N["dry-run-recorded selected action"]

    I --> O["RemediationDecision"]
    N --> O
    O --> P["Audit JSONL"]

    O --> Q{"Verification status recovered and decision not fallback?"}
    Q -->|yes| R["Append incident into incident history"]
    Q -->|no| S["No history update"]
```
