**Runtime Pipeline**
```mermaid
flowchart TD
    A["FastAPI / API caller"] --> B["run_static_pipeline(request, settings)"]
    B --> C["Load Settings from .env"]
    B --> D["Load RuntimeConfig from config/runtime.json"]
    D --> E["build_detectors(runtime_config, settings)"]
    B --> F["SQLiteIncidentStore"]
    B --> G["AiopsPipeline.run_once(metric_series)"]

    G --> H["Collector.collect()"]
    H --> I["Normalizer.normalize()<br/>CDO label/unit/window aliases"]
    I --> J["QualificationGate.evaluate()"]
    J --> K["FeatureBuilder.build()"]
    K --> L["DetectorEngine.evaluate()"]

    L --> L1["ThresholdDetector"]
    L --> L2["DependencyDetector"]
    L --> L3["NoDataDetector"]

    L1 --> M["CandidateEvent[]"]
    L2 --> M
    L3 --> M

    M --> N["Correlator.correlate()"]
    N --> O["Enricher.enrich(features)"]
    O --> P["SQLiteIncidentStore.upsert()"]
    P --> Q["Incident[]"]

    Q --> R["NotificationBuilder.build()"]
    Q --> S["PolicyEngine.proposal_for()"]
    S --> T["PolicyEngine.evaluate()"]
    T --> U["PolicyDecision[]"]

    Q --> V["VerificationEngine.verify(features)"]
    V --> W["VerificationResult[]"]

    G --> X["V001 RCA"]
    X --> Y["RcaResult"]

    Q --> Z["Remediation Strategy"]
    Y --> Z
    Z --> ZA["RemediationDecision[]"]
    ZA --> ZB["RemediationAuditLog.append()"]

    W --> ZC{"Verification recovered?"}
    ZA --> ZC
    Y --> ZC
    ZC -->|yes| ZD["IncidentHistoryStore.append_success()"]
    ZC -->|no| ZE["Do not update history"]

    R --> OUT["PipelineResult"]
    U --> OUT
    W --> OUT
    Y --> OUT
    ZA --> OUT
```

**Detector Flow**
```mermaid
flowchart TD
    A["Feature[]"] --> B["DetectorEngine"]
    B --> C["ThresholdDetector"]
    B --> D["DependencyDetector"]
    B --> E["NoDataDetector"]

    C --> C1{"feature ready and value > threshold?"}
    C1 -->|yes| C2["CandidateEvent reason=threshold_breached"]
    C1 -->|no| C3["No candidate"]

    D --> D1{"dependency signal ready and value > threshold?"}
    D1 -->|yes| D2["CandidateEvent reason=dependency_signal_breached likely_dependency=..."]
    D1 -->|no| D3["No candidate"]

    E --> E1{"required signal status unknown?"}
    E1 -->|yes| E2["CandidateEvent reason=signal_missing/stale/invalid"]
    E1 -->|no| E3["No candidate"]

    C2 --> F["CandidateEvent[]"]
    D2 --> F
    E2 --> F
```

**RCA Pipeline**
```mermaid
flowchart TD
    A["metric_series from request"] --> B{"RCA enabled and metric_series exists?"}
    B -->|no| C["RcaResult empty"]
    B -->|yes| D["V001AnomalyEngine.evaluate()"]

    D --> D1["EwmaStlDetector"]
    D --> D2["ServiceIsolationForestDetector"]

    D1 --> E["AnomalyFinding[]"]
    D2 --> E

    E --> F["V001RcaEngine.rank(findings, metric_series, top_k)"]
    A --> F

    F --> G["GraphTraversalRca.rank_services(findings)"]
    F --> H["RobustScoreRca.rank(metric_series, anomaly_timestamp)"]

    G --> I["service_scores from topology dependencies"]

    I --> K["Merge service scores"]

```

**Remediation Pipeline**
```mermaid
flowchart TD
    A["Incident + RcaResult"] --> B["RemediationFeatureExtractor.extract()"]

    B --> B1["affected_services = incident.service + dependency + RCA services"]
    B --> B2["log_signatures = event.reason"]
    B --> B3["metric_ratios = value / threshold"]

    B1 --> C["IncidentFeatures"]
    B2 --> C
    B3 --> C

    C --> D["IncidentHistoryStore.load()"]
    C --> E["HistoryRetriever.top_matches()"]

    D --> E
    E --> E1["similarity = service jaccard + log jaccard + metric similarity"]

    E1 --> F["RemediationDecisionEngine.decide()"]
    G["ActionCatalog.load()"] --> F

    F --> H{"similarity < OOD threshold?"}
    H -->|yes| I["fallback page_oncall"]
    H -->|no| J["Vote actions from matched history"]

    J --> K{"action exists in catalog?"}
    K -->|no| I
    K -->|yes| L["Compute confidence + expected_cost"]

    L --> M{"Guardrails pass?"}
    M -->|no| I
    M -->|yes| N["dry-run-recorded selected action"]

    I --> O["RemediationDecision"]
    N --> O
    O --> P["Audit JSONL"]

    O --> Q{"Verification status recovered and decision not fallback?"}
    Q -->|yes| R["Append incident into incident history"]
    Q -->|no| S["No history update"]
```
