**4. Feature builder**

Feature builder biến observation thô thành feature có ý nghĩa vận hành.

Nó tạo các nhóm feature như:

- Official rolling 24h SLO:
  - checkout success ratio;
  - cart success ratio;
  - storefront p95;
  - browse/search availability.
- Diagnostic 5m/15m:
  - QPS;
  - error ratio;
  - p50/p95/p99;
  - DB backend usage.
- Robust anomaly baseline:
  - median;
  - MAD;
  - EWMA;
  - robust z-score.
- Freshness/sample count:
  - last sample time;
  - total requests;
  - no traffic vs no data.

Điểm quan trọng:

> 5m/15m chỉ là diagnostic hoặc anomaly input, không thay thế official rolling 24h SLO.
